## 🌟 About the Project

SkalGPT is a versatile AI assistant specifically developed for Sezai Karakoç Anatolian High School students, teachers, and staff. Built with modern web technologies, this platform stands out with its user-friendly interface and robust infrastructure.

### 🎯 Key Features

- 📚 Academic support and resource management
- ✍️ Creative writing and content generation
- 🔍 Text summarization and analysis tools
- 🧮 Problem solving and calculations
- 🌍 Multi-language support (i18n)
- 🌓 Dark/Light theme support
- 📱 Responsive design

## 🔧 Installation

To run the project locally, follow these steps:

1. **Clone the repository**
   ```bash
   git clone https://github.com/yourusername/skalgpt-website.git
   cd skalgpt-website
   ```

2. **Install dependencies**
   ```bash
   npm install
   # or
   yarn install
   # or
   pnpm install
   ```

3. **Start the development server**
   ```bash
   npm run dev
   # or
   yarn dev
   # or
   pnpm dev
   ```

4. **Open in browser**
   The application will be available at: [http://localhost:3000](http://localhost:3000)

## 🧠 Technologies Used

- **Frontend Framework:** Next.js 14
- **Language:** TypeScript
- **Styling:** Tailwind CSS
- **UI Components:** Radix UI, Shadcn/ui
- **State Management:** React Context API
- **Form Management:** React Hook Form
- **Internationalization:** next-i18next
- **Icons:** Lucide Icons
- **Deployment:** Netlify

## 📦 Customization

### Editing Translations

1. Edit the relevant language file in the `locales/` directory.
2. To add a new language, create a new JSON file and add it to the `i18n.js` file.

### Changing Theme Colors

1. Edit the `theme.extend.colors` section in `tailwind.config.js`.
2. Update CSS variables in `styles/globals.css` for custom colors.

### Adding New Components

1. Create a new component in the `components/ui/` directory.
2. Add the component to the `components/index.ts` file.

## 📦 Özelleştirme

### Çevirileri Düzenleme

1. `locales/` dizinindeki ilgili dil dosyasını düzenleyin.
2. Yeni bir dil eklemek için yeni bir JSON dosyası oluşturun ve `i18n.js` dosyasına ekleyin.

### Tema Renklerini Değiştirme

1. `tailwind.config.js` dosyasındaki `theme.extend.colors` bölümünü düzenleyin.
2. Özel renkler için `styles/globals.css` dosyasında CSS değişkenlerini güncelleyin.

### Yeni Bileşen Ekleme

1. `components/ui/` dizininde yeni bir bileşen oluşturun.
2. Bileşeni `components/index.ts` dosyasına ekleyin.
